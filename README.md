## 👤 关于 ADui Studio

**ADui Studio ≠ 团队 ≠ 公司**

它只是：
- 我自己的一个 **项目收纳盒**
- 一个用来做实验、踩坑、重构的地方
- 一些“可能没啥用，但我觉得有意思”的代码集合

如果你在这里看到：
- 半成品
- 重构中的项目
- README 写一半的仓库  

那都很正常。

## 🧩 这里通常会有什么
- 前端实验项目
- 全栈练习 / Side Project
- 工程化尝试
- 小工具 / 奇怪想法的实现
- 偶尔比较认真的开源项目

## 📌 使用说明
- 项目风格不统一（因为我还在成长）
- 代码质量不稳定（但会越来越好）
- Commit 信息基本是真实心情记录

## 🧠 一点态度
我不太相信“一开始就完美的代码”。

更相信：
> 写完 → 推翻 → 重写 → 再推翻

**ADui Studio** 存在的意义，  
就是让我可以心安理得地反复折腾。
